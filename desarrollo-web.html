Esto es un proceso complejo que implica la creación de productos digitales en línea. Para conseguirlo se hace uso de tecnologías de software del lado del servidor y del cliente que involucran una combinación de proceso de base de datos con el uso de un navegador web a fin de realizar determinadas tareas o mostrar información.

Lo que aprenderás

-Construir sitios y aplicaciones web

-Trabajar como desarrollador Web Junior

-Comenzar su propio negocio online

-Trabajar con bases de datos y leguajes del lado del servidor

-Desarrollar aplicaciones móviles basadas en HTML

-Trabajar por proyectos en sitios de trabajo Freelance

-Desarrollar sitios webs con confianza
